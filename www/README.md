# www

## Quickstart

Run:
```
npm install
npm start
# open http://localhost:3000
```

Debug:
```
DEBUG=www npm start
```

## Routes

# Home
`/`

# Search
`/?q=merge+sort`

# List of algorithms
`/algorithms`

# Search by language
`/?lang=python`

# Algorithm page
`/{algorithmId}`

i.e. `/factorial`
